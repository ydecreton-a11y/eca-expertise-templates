# 07 — Doctrine Administrative (BOFiP)

## Organisation par impôt

### IS — BOI-IS-*
- BOI-IS-BASE : base imposable
- BOI-IS-GPE : intégration fiscale
- BOI-IS-RICI : régimes d'imposition et crédits d'impôt

### IR — BOI-IR-*
- BOI-IR-PAS : prélèvement à la source
- BOI-IR-RICI : réductions et crédits d'impôt
- BOI-RPPM-PVBMI : plus-values mobilières
- BOI-RFPI-PVI : plus-values immobilières

### TVA — BOI-TVA-*
- BOI-TVA-CHAMP : champ d'application
- BOI-TVA-IMM : TVA immobilière

### Enregistrement — BOI-ENR-*
- BOI-ENR-DMTG : droits de mutation à titre gratuit
- BOI-ENR-DMTOI : mutations à titre onéreux d'immeubles

## Méthode d'utilisation
Vérification systématique via MCP Légifrance avant toute citation en consultation.
Lien direct : https://bofip.impots.gouv.fr/bofip/

# 08 — Jurisprudence

## Organisation

### Conseil d'État (CE)
Contentieux fiscal — recours en cassation
- `Conseil-Etat/`

### Cour de cassation — Chambre commerciale (Cass.com)
Droit des sociétés, cessions, garanties de passif
- `Cour-de-Cassation/`

### Cours d'appel
Jurisprudence de fond — Douai, Amiens, Paris principalement
- `Cours-Appel/`

## Format de référencement
```
[Juridiction], [date], n°[numéro de pourvoi/arrêt]
Thème : [résumé 1 ligne]
Portée : [impact pratique]
Lien Légifrance : [URL]
```

## Vérification
Toutes les références jurisprudentielles sont vérifiées via MCP Légifrance avant intégration.

# 📁 ECA EXPERTISE — Bibliothèque de modèles juridiques

Bibliothèque de modèles d'actes juridiques et fiscaux du cabinet **ECA EXPERTISE** (Cambrai · Arras · Valenciennes).

## Structure

| Dossier | Contenu |
|---|---|
| `01_Statuts` | Statuts SAS, SARL, SARL holding, SCI, SELARL, SPFPL |
| `02_Cessions_Titres` | Cessions d'actions SAS, SELAS, parts SARL, SCI, SCM, patientèle |
| `03_Fonds_Commerce` | LOI, compromis, actes définitifs cession fonds |
| `04_MA_Exotel` | Séquence M&A complète — Exotel (LOI → Protocole → Acte → GAP) |
| `04_MA_ADEQUA` | Séquence M&A complète — ADEQUA cabinet EC |
| `04_MA_Garages_Lucas_DLS` | Séquence M&A complète — Garages Lucas / DLS |
| `05_PV_Decisions` | PV AGE, DU dissolution, DU liquidation, DU distribution dividende |
| `06_Apport_Holding` | Traité d'apport nature, CAC apports, Attestation 150-0 B ter |
| `07_Sortie_Associe_Liquidation` | Convention crédit-vendeur, abandon compte courant |
| `08_Pacte_Associes` | Modèle de pacte SAS (42 articles) + clauses sanctions |
| `09_Conventions` | Convention de commissionnement intra-groupe (prix de transfert) |
| `10_Operationnel_Immatriculation` | Devis ECA (template charte), procuration, attestations |
| `11_Consultations` | Consultations fiscales (ex. filiale belge + entrepôt France) |

# ✅ CHECKLIST : JOUR DE LA TRANSMISSION

À utiliser **lors de la finalisation de l'acte** et **avant dépôt DMTG**.

---

## 📋 PHASE IMMÉDIATE (À la signature)

### Vérifications finales dans l'acte

- [ ] **Acte signé** (date précisée : **≥ ou < 21/02/2026** ?)
- [ ] **ECU présent** (si stratégie choix ECU préalable)
- [ ] **EIC présent** (obligatoire, signé par bénéficiaire)
  - [ ] Durée correcte : 6 ans (≥ 21/02/2026) ou 4 ans (< 21/02/2026)
  - [ ] Formule expresse
- [ ] **Identification bénéficiaires**
  - [ ] Noms prénoms exacts
  - [ ] Nombre/valeur titres précis
  - [ ] Statut (enfant, conjoint, etc.) pour calcul abattements

### Société : Documentation immédiate

**À demander au gérant / Président / Expert-comptable** :

- [ ] **Attestation notariale de la société** :
  ```
  ✓ Nombre exact titres transmis
  ✓ Valeur vénale estimée (base DMTG)
  ✓ Composition actif brut (%)
  ✓ Titres composés (% actifs opérationnels vs patrimoine)
  ✓ Identité dirigeant exercant fonction
  ✓ Exercice effectif fonction depuis [date]
  ✓ Respect seuils ECU : [%] capital + [%] droits de vote
  ✓ Biens somptuaires identifiés (si transmission ≥ 21/02/2026)
  ```

- [ ] **Copie statuts actuels** (post-modification statuts le cas échéant)

- [ ] **PV de direction** (dernière année)
  - [ ] Attestant exercice fonction par signataire ECU

- [ ] **Comptabilité/Trésorerie**
  - [ ] Derniers comptes (pour valorisation)
  - [ ] Actifs figés (liste, valeur estimée)

### Cas spéciaux : Documents additionnels

#### 🔹 Si démembrement NP/Usufruit

- [ ] **Modification statuts antérieure** (attestée)
  - [ ] Droits de vote usufruitier limités à "affectation bénéfices"
  - [ ] Date publication (JAL, BODACC)
  - [ ] Copie PV modification en annexe acte

- [ ] **Justificatif âge de l'usufruitier** (pour barème CGI 669)
  - [ ] Acte de naissance ou carte identité
  - [ ] Nombre d'années calculé au jour transmission

#### 🔹 Si interposition (1 ou 2 niveaux)

- [ ] **Holding(s) interposée(s) documentée(s)**
  - [ ] Statuts
  - [ ] Composition actif (% participations Cible)
  - [ ] Respect test : participations > 50 % actif brut

- [ ] **Formule exonération calculée**
  - [ ] 1 niveau : 75 % × Valeur × (B₁/C₁)
  - [ ] 2 niveaux : Double calcul documenté

#### 🔹 Si apport à holding (durant EIC)

- [ ] **Holding réceptrice identifiée**
  - [ ] Copie statuts
  - [ ] Contrôle ≥ 75 % % des signataires ECU/bénéficiaires EIC
  - [ ] Composition actif > 50 % participations opérationnelles

- [ ] **Acte d'apport** daté et signé (si apport antérieur)

#### 🔹 Si biens somptuaires présents (LF 2026)

- [ ] **Inventaire biens somptuaires** : 
  ```
  Description | Valeur estimée | Affecté activité ? | Doc
  ─────────────────────────────────────────────────────
  Yacht       | 150 k€        | Non               | Facture
  Voiture     | 80 k€         | Oui (service)     | Contrat
  ...
  ```

- [ ] **Justification affectation** (si bien déclaré "affecté")
  - [ ] Contrat de mise en location/exploitation
  - [ ] Facturation client générée
  - [ ] Durée affectation ≥ 3 ans avant transmission ✓
  - [ ] Couverture jusqu'à fin EIC ✓

- [ ] **Fraction d'exclusion calculée**
  ```
  = Σ(Valeur biens somptualres non-affectés) / Actif brut × Valeur transmission
  ```

---

## 📑 DÉCLARATION DMTG : PIÈCES À ASSEMBLER

### Pièces obligatoires

- [ ] **Déclaration DMTG complétée** (Cerfa 2735)
  - [ ] Identité donateur/défunt + bénéficiaires
  - [ ] Valeur vénale (attestation société)
  - [ ] Calcul DMTG avec application exonération 75 %
  - [ ] Références engagements ECU/EIC

- [ ] **Acte de transmission** (copie intégrale)
  - [ ] Donation notariée (minute) OU
  - [ ] Succession (testament + acte liquidation) OU
  - [ ] Apport-cession (acte sous-seing privé avec date certaine)
  - [ ] Signé et daté par toutes parties

- [ ] **Copie engagement de conservation**
  - [ ] ECU (si applicabe)
  - [ ] EIC (obligatoire)
  - [ ] Signé par bénéficiaire

- [ ] **Attestation de la société**
  - [ ] Modèle conseil notarial
  - [ ] Signée gérant/Président/Expert-comptable

### Pièces complémentaires (selon situation)

#### 📎 Démembrement NP/Usufruit :
- [ ] Copie modification statuts (PV + publication)
- [ ] Justificatif âge usufruitier
- [ ] Calcul abattement 669 (barème)

#### 📎 Interposition :
- [ ] Statuts holding(s)
- [ ] Tableau composition actif holding
- [ ] Formule calcul exonération double-niveau (si 2 niveaux)

#### 📎 Biens somptuaires (≥ 21/02/2026) :
- [ ] Inventaire biens somptualres
- [ ] Justificatifs affectation (contrats, factures)
- [ ] Calcul fraction exclue

#### 📎 Apport à holding en cours EIC :
- [ ] Acte apport + date
- [ ] Statuts holding (preuve ≥ 75 % + > 50 % actif)

## 🎯 POINTS CRITIQUES DE VÉRIFICATION

### 🔴 Risques à 100 % neutraliser

- [ ] **Date transmission documentée** avec précision
  - [ ] < 21/02/2026 → EIC 4 ans
  - [ ] ≥ 21/02/2026 → EIC 6 ans
  - Une seule date possible : **la date d'acte définitif**

- [ ] **EIC signature bénéficiaire** présente dans acte
  - [ ] Impossible à rattraper après → exonération perdue rétroactivement

- [ ] **Statut bénéficiaire** confirmé (enfant/conjoint/tiers)
  - [ ] Impact calcul abattements (déduction OR droit de vote)

- [ ] **Seuils ECU respectés** si ECU signé
  - [ ] 17 % capital + 34 % droits vote (non-côté)
  - [ ] Sinon : exonération non applicable

### 🟠 Risques importants

- [ ] **Fonction direction exercée** avant ET après transmission
  - [ ] Signataire ECU : ≥ 2 ans avant
  - [ ] Bénéficiaire EIC : ≥ 3 ans après (ou signataire ECU)
  - Attestation PV : À conserver

- [ ] **Activité opérationnelle confirmée**
  - [ ] ❌ Pas LMNP depuis 17/10/2023
  - [ ] ❌ Pas gestion patrimoine pure

- [ ] **Démembrement : droits vote usufruitier limités**
  - [ ] Statuts modifiés AVANT transmission
  - [ ] Publication attestée
  - Sinon : exonération perdue sur usufruit

### 🟡 Points d'attention

- [ ] **Biens somptuaires** (si transmission ≥ 21/02/2026)
  - [ ] Inventaire exhaustif
  - [ ] Justification affectation (si prétendue)
  - [ ] Calcul d'exclusion fait

- [ ] **Interposition** : Respect nb niveaux (max 2)

- [ ] **ECU signé avant** transmission (pas rétroactif)

## 📐 CALCULS À VÉRIFIER

### Exonération simple (1 niveau, pas démembrement)

```
DMTG brut = Valeur titres × Tarif DMTG applicale (5 à 60%)
DMTG après Dutreil = Valeur titres × 25% × Tarif DMTG
(ou 0 si tarif ≤ 20%)

Vérifier : Application abattements bénéficiaire
(enfant : 100 k€, conjoint : 80 k€, si éligible)
```

### Exonération avec biens somptuaires (LF 2026)

```
Valeur assiette = Valeur titres - (Valeur biens somptualres non-affectés / Actif brut × Valeur titres)

DMTG = (Valeur assiette) × 25% × Tarif
```

### Démembrement NP + Usufruit

```
1. Valorisation NP = Valeur titres × % NP (barème 669, âge usufruitier)
2. Valorisation Usufruit = Valeur titres × % Usufruit
3. Chaque fraction → abattement 669 + exonération 75%
4. DMTG = [NP réduite + Usufruit réduite] × 25% × Tarif
```

## ✉️ DÉPÔT ADMINISTRATIF

### Où déposer

- [ ] **Notaire** (donation notariée)
- [ ] **Tribunal instance** (succession)
- [ ] **SIE (Services Impôts Entreprise)** si transmission interne à groupe

### Délais

- [ ] **Donation** : 1 mois de la signature notariale
- [ ] **Succession** : 6 mois du décès
- [ ] **Apport-cession** : Dans le mois du dépôt comptes

### Contrôle administratif post-dépôt

- [ ] **Administration peut demander** (30 jours de réponse)
  - [ ] Confirmation respect engagements
  - [ ] Attestation direction
  - [ ] Détails biens somptuaires

- [ ] **Garder dossier complet** ≥ 6 ans (prescription DMTG)

---

## 🚨 ERREURS CLASSIQUES À ÉVITER

| Erreur | Conséquence | Prévention |
|---|---|---|
| EIC oublié dans acte | Exonération perdue rétro | Vérifier formule expresse |
| Date transmission imprécise | Doute EIC 4 vs 6 ans | Date acte définitif seulement |
| ECU expiré avant transmission | ECU non-valable | Dater ECU : au minimum 2 ans avant transmission |
| Statuts NP non-modifiés avant | Usufruit non-exonéré | Modifier AVANT donation |
| Dirigeant change, vacance | Déchéance direction | Assurer relais dirigeant, doc PV AG |
| LMNP incluse (depuis 17/10/2023) | Exonération perdue | Vérifier activité = non-LMNP |
| Biens somptualres non-documentés | Fraction indûment exonérée | Inventaire + justificatif affectation |
